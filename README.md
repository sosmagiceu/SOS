# S.O.S. Magic — Astro rebuild

Modern rebuild using Astro, React and Tailwind CSS. The original page copy, media,
links, forms, legal text and interactive behaviour are preserved in full.

## Start locally

```bash
npm install
npm run dev
```

## Production

```bash
npm run build
npm run preview
```

The existing Netlify contact and review functions are retained. Configure the same SMTP and Supabase environment variables in Netlify before deployment.
